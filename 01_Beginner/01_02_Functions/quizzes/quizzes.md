# Beginner Quiz – Functions
1. What keyword starts a function definition?  
   a) func  b) def  c) function  d) fn
2. What does a function return if there is no `return` statement?  
   a) 0  b) ""  c) None  d) Error
3. In `def greet(name="Guest"):` what is "Guest"?  
   a) argument  b) default parameter  c) keyword  d) variable
4. Can a function have zero parameters?  
   a) Yes  b) No
**Answers**  
1-b, 2-c, 3-b, 4-a



# Intermediate Quiz – Functions
1. `*args` collects arguments as:  
   a) list  b) tuple  c) dict  d) set
2. `**kwargs` collects arguments as:  
   a) list  b) tuple  c) dict  d) set
3. Which rule describes variable lookup order?  
   a) LEGB  b) GEBL  c) LGEB  d) ELGB
4. Lambda functions are also called:  
   a) named functions  b) anonymous functions  c) static functions  d) global functions
**Answers**  
1-b, 2-c, 3-a, 4-b




# Advanced Quiz – Functions
1. `nonlocal` is used to modify variable in:  
   a) global scope  b) enclosing scope  c) built-in scope  d) local scope
2. Which function is most commonly used with lambda?  
   a) print()  b) len()  c) sorted() / map() / filter()  d) type()
3. A function that returns another function is called:  
   a) closure factory  b) higher-order function  c) decorator  d) generator
4. Type hints were introduced in Python version:  
   a) 3.5  b) 3.6  c) 3.7  d) 3.9
**Answers**  
1-b, 2-c, 3-b, 4-a
