# Beginner Quiz - Variables, Data Types & Operators

1. Which of these is mutable?  
   a) int  b) str  c) list  d) tuple  
   **Answer: c) list**

2. What is the output of `10 // 3`?  
   a) 3  b) 3.333  c) 1  d) Error  
   **Answer: a) 3** (floor division)
3. `type(3.14)` returns?  
   **Answer: <class 'float'>**
4. Can you change the value of a variable in Python?  
   **Answer: Yes** (dynamic typing)
5. What does `%` operator do?  
   **Answer: Modulo (remainder)**

**Answer Key with Explanation**:- 
Q1: List is mutable, others are immutable.- 
Q2: `//` always gives floor result.- 
Q3: 3.14 is floating point.- 
Q4: Python variables can hold any type anytime.- 
Q5: `10 % 3` = 1 (remainder).
